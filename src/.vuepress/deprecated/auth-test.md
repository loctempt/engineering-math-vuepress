## 说明

- **登录功能**: 点击"登录"按钮打开登录弹窗
- **状态管理**: 登录状态会在页面刷新后保持
- **用户信息**: 显示当前登录用户的姓名、邮箱和角色
- **退出登录**: 可以安全退出当前会话

## 技术特性

- 使用 Vue 响应式状态管理
- 集成 Waline API 进行认证
- 支持管理员和普通用户角色
- 本地存储会话持久化

::: tip
当前仅支持邮箱密码登录，第三方登录功能待配置。
:::
